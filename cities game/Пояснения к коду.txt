
1. Голосовой ввод реализован с помощью 
   технологии SpeechKit JavaScript API 

2. Отображение городов на карте реализовано с помощью
   гехнологии геокодирования API Яндекс.Карт

3. Карта представляет собой объект API Яндекс.Карт

4. До начала игры можно запросить за один раз 100 случайных городов,
   чтобы пользователь перед игрой мог освежить свои знания.
   Во время игры такой возможности нет.

5. База городов имеет примерно 24000 городов.
